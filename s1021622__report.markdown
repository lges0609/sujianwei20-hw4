**Clock Algorithm **
=======================================================
-------------------------------------------------------

##***What is Clock Algorithm ???***
**Clock Algorithm **是**LRU algorithm **的一種近似演算法,**LRU algorithm** 雖然有許多優點
但它在實做上卻是困難重重,利用**Clock Algorithm** 來近似**LRU algorithm** 增加實做可行性
也增加演算法速度.
